## How to Track Your Contributions

1. **Fork** the repository
2. **Create feature branch**: `feature/your-contribution`
3. **Make changes** with clear commit messages
4. **Submit PR** with description
5. **Get merged** and added to this tracker!

---

## Submission Template

When submitting a PR, include:

```markdown
## Contribution Type
- [ ] Typo/Grammar fix
- [ ] Code example
- [ ] Exercise
- [ ] Content enhancement
- [ ] New guide
- [ ] Example project
- [ ] Other: _______

## Description
What did you add/improve?

## Related Issue
Fixes #(issue number)

## Points Requested
Based on type: __ points
```
